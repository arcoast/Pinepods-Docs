# Start Page Settings

The Start Page Settings feature allows users to customize which page they land on immediately after logging into Pinepods. This personalization option helps users streamline their workflow by starting directly on their most frequently used page.

## Overview

Instead of always landing on the default home page after login, users can choose to start on any of the main Pinepods pages. This setting is saved to your user profile and syncs across all devices and platforms where you use Pinepods.

## Accessing Start Page Settings

1. Navigate to **Settings** in the main menu
2. Expand the **Start Page Settings** section
3. All users can modify their personal start page preference

## Available Start Page Options

### Main Pages
You can set any of these pages as your start page:

- **Home**: The default dashboard with overview information
- **Feed**: Your main podcast episode feed with latest episodes
- **Search**: The podcast search interface for discovering new content
- **Queue**: Your episode queue for planned listening
- **Saved**: Your saved/bookmarked episodes
- **Downloads**: Your downloaded episodes for offline listening
- **People Subscriptions**: Episodes from people you follow
- **Podcasts**: Your podcast subscription library

### Choosing the Right Start Page

**Home**
- Best for: Users who want an overview of their podcast activity
- Features: Dashboard view, recent activity, recommendations

**Feed** 
- Best for: Users who immediately want to see new episodes
- Features: Latest episodes from subscribed podcasts, chronological order

**Search**
- Best for: Users who frequently discover new podcasts
- Features: Podcast discovery, search functionality, trending content

**Queue**
- Best for: Users who plan their listening sessions in advance
- Features: Organized listening queue, episode management

**Saved**
- Best for: Users who bookmark episodes for later
- Features: Personal episode collection, saved content

**Downloads**
- Best for: Users who primarily listen to downloaded content
- Features: Offline episode management, download status

**People Subscriptions**
- Best for: Users who follow specific podcast hosts or creators
- Features: Person-based content discovery, creator following

**Podcasts**
- Best for: Users who manage their podcast subscriptions actively
- Features: Subscription management, podcast library organization

## How to Change Your Start Page

### Step-by-Step Process
1. Open the **Start Page Settings** section
2. Wait for the current setting to load (loading spinner will disappear)
3. Click the dropdown menu to see all available page options
4. Select your preferred start page from the list
5. Click **Apply** or **Save** to confirm your choice
6. The change takes effect on your next login

### Immediate Application
- Changes are saved both locally and to your user profile
- New start page applies immediately to future login sessions
- Setting syncs across all your Pinepods applications

## Cross-Platform Synchronization

### Device Sync
Your start page preference is synchronized across:
- Web browser interface
- Desktop applications
- Mobile apps (iOS and Android)
- Any other Pinepods clients

### Profile Integration
- Setting is tied to your user account, not device-specific
- Logging in from any new device will use your saved preference
- Changes made on one device immediately affect all others

## Technical Details

### Storage and Sync
- **Local Storage**: Immediate application using browser storage
- **Server Storage**: Permanent setting saved to user profile
- **Cross-Platform**: Setting syncs via server-side user data

### Session Behavior
- **Login Redirect**: Automatic redirect to chosen page after authentication
- **Session Persistence**: Setting maintained throughout user session
- **Logout Reset**: Returns to login page when session ends

### Performance
- **Fast Loading**: Chosen page loads immediately after authentication
- **Resource Efficiency**: No unnecessary page loads or redirects
- **Seamless Transition**: Smooth user experience from login to content

## Troubleshooting

### Start Page Not Changing
- **Clear Browser Cache**: Old cached data may interfere
- **Verify Save**: Ensure you clicked save/apply after selecting
- **Check Network**: Stable connection required to sync setting
- **Try Different Browser**: Test if issue is browser-specific

### Setting Not Syncing Across Devices
- **Wait for Sync**: Changes may take a moment to propagate
- **Force Refresh**: Log out and back in on other devices
- **Check Connection**: Ensure all devices have internet access
- **Account Verification**: Confirm you're using the same account

### Page Loading Issues
- **Permissions**: Ensure you have access to the selected start page
- **Content Availability**: Some pages require existing content to display
- **Server Status**: Check if Pinepods server is responding normally
- **Fallback**: System may default to Home page if start page fails
